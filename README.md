# LabProgrammazione
Progetto laboratorio di programmazione 2026 Java

GitHub è una piattaforma che permette di gestire e condividere codice usando Git, un sistema di controllo versione. Con comandi come git clone scarichi il progetto, git add e git commit salvi le modifiche, git push le carichi online e git pull aggiorni il codice con le ultime versioni. In questo modo più persone possono collaborare in modo ordinato sullo stesso progetto.
